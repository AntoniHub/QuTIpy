Quantum Theory of Information for Python; pronounced `/cutiɛ paɪ/`.
A package for performing calculations with quantum states, channels and protocols.
It is comparable to the `QETLAB package <http://www.qetlab.com/Main\_Page>`_ for MATLAB/Octave.
